# Contributing

Contributions that improve reproducibility, documentation, testing, or validation methodology are welcome.

1. Open an issue describing the proposed change.
2. Create a focused branch.
3. Run `python verify_precomputed.py` and `pytest -q`.
4. For scientific changes, run `python analysis.py` and document every changed result.
5. Submit a pull request with the rationale, affected files, and any interpretation changes.

Do not add proprietary data, confidential media formulations, credentials, patient data, or files whose redistribution terms are unclear.
