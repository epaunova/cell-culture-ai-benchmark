# Precomputed outputs

These files are regenerated by `python analysis.py`.

- `benchmark_metrics.csv` - model metrics for each task, model, and validation protocol.
- `external_validation_metrics.csv` - performance on the 16 held-out four-factor conditions.
- `pbmc_homeostasis_scores.csv` - non-compensatory PBMC formulation ranking.
- `kp4_pareto_front.csv` - efficient productivity/process-load candidates.
- `results_summary.json` - complete machine-readable result object, including fold summaries and predictions.

Run `python verify_precomputed.py` for a fast integrity check.
